# Task 3: Linear Regression

## Objective
Implement and understand Simple and Multiple Linear Regression using Scikit-Learn.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-Learn

## Dataset
Housing Price Prediction Dataset

## Procedure
1. Imported and preprocessed the dataset.
2. Split the data into training and testing sets.
3. Implemented Simple Linear Regression.
4. Implemented Multiple Linear Regression.
5. Evaluated models using MAE, MSE, and R² Score.
6. Visualized the regression line and interpreted coefficients.

## Results

### Simple Linear Regression
- MAE: 1,474,748.13
- MSE: 3,675,286,604,768.19
- R² Score: 0.2729

### Multiple Linear Regression
- MAE: 1,265,275.67
- MSE: 2,750,040,479,309.05
- R² Score: 0.4559

## Conclusion
Multiple Linear Regression performed better than Simple Linear Regression because it used multiple features to predict house prices, resulting in a higher R² score and lower error values.

## Learning Outcomes
- Learned data preprocessing techniques.
- Understood Simple and Multiple Linear Regression.
- Evaluated model performance using MAE, MSE, and R² Score.
- Visualized regression results using Matplotlib.
